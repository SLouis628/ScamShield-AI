# ScamShield AI

ScamShield AI is an AI-powered scam detection platform designed to help users identify fraudulent text messages, suspicious links, phishing attempts, and scam phone numbers.

## Features

- Keyword-based scam detection
- Suspicious URL detection
- Threat scoring system
- Risk classification (Low, Medium, High)
- AI-generated explanations
- Future phone number reputation lookup
- Future domain reputation analysis
- Future scam database integration
