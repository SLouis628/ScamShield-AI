ScamShield AI Project Proposal

Project Title

ScamShield AI: An Explainable AI-Based Mobile Threat Detection System for SMS Scam and Phone Number Analysis

Problem Statement

SMS phishing attacks (smishing) continue to increase and target users through fraudulent text messages, malicious links, and impersonation scams. Many users struggle to determine whether a message is legitimate before interacting with it.

Project Objective

The objective of ScamShield AI is to develop a mobile threat detection platform capable of analyzing suspicious text messages, phone numbers, and embedded links using artificial intelligence and explainable threat analysis.

Key Features

Option C - Share to ScamShield

- Share suspicious SMS messages directly to ScamShield
- Instant threat analysis
- URL inspection
- Phone number reputation scoring

Option B - ScamShield Messenger

- Android SMS application
- Built-in Analyze Threat button
- Real-time threat detection
- Message history and threat history

Technologies

- Python
- FastAPI
- Scikit-Learn
- Kotlin
- Android Studio
- SQLite
- Git
- GitHub

Expected Outcome

ScamShield AI will help users identify potentially fraudulent text messages and educate users on common scam tactics through explainable AI-generated threat assessments.
